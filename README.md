# College Dost 🚀

## 📌 Overview

**College Dost** is a student support and mentorship platform designed to solve common problems faced by engineering students.
The project helps students get guidance, mentorship, career roadmaps, doubt-solving support, and real-time communication with mentors and alumni.

This project was developed as a mini project using full-stack web development technologies.

---

# 🎯 Problem Statement

When students enter engineering, many of them face problems such as:

* Lack of proper career guidance
* Confusion about engineering exams and academics
* No clear roadmap for skills and placements
* Hesitation in asking doubts to teachers or seniors
* Lack of industry exposure and updated skills
* Personal and mental health-related concerns
* Difficulty understanding real-world career requirements

To solve these problems, **College Dost** was developed as a platform where students can:

* Get mentorship support
* Ask doubts through chat systems
* Use anonymous chat for sensitive questions
* Learn from alumni experiences
* Follow roadmap-based career guidance

---

# ✨ Main Features

## 🔐 Authentication System

* Student Signup/Login using PRN
* Mentor Login
* Alumni Login
* Admin Dashboard Management

## 💬 Real-Time Chat System

* One-to-one student and mentor communication
* Built using **Socket.IO**

## 🕵️ Anonymous Chat Feature

* Students can ask personal/sensitive doubts anonymously
* Mentor identity remains visible

## 👨‍🏫 Mentor Support

* Mentors help students with technical and non-technical guidance
* Mentor support system is used for special mentor recognition.

## 🎓 Alumni Spotlight

* Alumni share:

  * Industry experience
  * Career guidance
  * Workplace knowledge
  * Internship and placement tips

## 🛣️ Roadmap Generator

* Semester-wise roadmap based on selected domain
* Helps students learn required technologies step-by-step

## 📢 Announcement System

* Workshops
* Events
* Internship opportunities
* Job opportunities
* Technical and non-technical activities

## ⚙️ Admin Dashboard

* Add / Update / Delete users
* Manage students, mentors, and alumni
